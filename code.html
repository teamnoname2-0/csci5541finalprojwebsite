Code & Resources

All code for our fake job detection system is publicly available. Below you'll find links to the Chrome extension source code, the backend API, and our training notebooks.

Chrome Extension

Extension Frontend (GitHub)

The Chrome extension source code, including the content scripts for scraping job postings and the popup UI that displays fraud risk analysis results.

View on GitHub

Backend API (Hugging Face Spaces)

The FastAPI backend hosted on Hugging Face Spaces. Handles text parsing via GPT-4o-mini, runs the three-model ensemble, computes evidence spans, and returns predictions.

View on Hugging Face

Training Notebooks

LUKE Model Training

Dataset merging and preprocessing, fine-tuning the LUKE model on the combined job posting datasets, training loop with evaluation metrics, and counterfactual fairness analysis

Open in Google Colab

RoBERTa & DistilBERT Training

Notebook for fine-tuning RoBERTa and DistilBERT using the same dataset and procedure as LUKE, completing the three-model ensemble.

Open in Google Colab
